📖 Guide d'utilisation

Démarrer une partie

  1. Allez dans Joueurs et créez vos 4 profils (nom + photo optionnelle).
  2. Depuis l'Accueil, appuyez sur Nouvelle Partie.
  3. Sélectionnez les 4 joueurs participant à la partie.
  4. Appuyez sur Lancer la partie.

Saisir les scores d'une manche

Sur l'écran de jeu, le donneur actuel est mis en évidence. Appuyez sur un contrat disponible pour saisir les scores :

  • Barbu / Dernier Pli — sélectionnez le joueur qui l'a pris.
  • Pas de Dames / Plis / Cœurs — ajustez le compteur pour chaque joueur.
  • La Salade — saisie combinée des 4 sous-contrats.
  • La Réussite — classez les joueurs par ordre de sortie.

Validez avec Confirmer. Le tableau des scores est mis à jour automatiquement.

🥗 La SaladeCombinaison des 4 contrats ci-dessusTotal réparti
🌟 La RéussiteRéussir à se défausser en premierBonus décroissant par rang

Modifier / corriger une manche

Dans l'écran de jeu, ouvrez l'Historique des manches et appuyez sur une ligne pour l'éditer. Les scores sont recalculés automatiquement.

Paramètres des scores

Accédez à Règles pour personnaliser les valeurs de chaque contrat selon vos règles maison. Les modifications s'appliquent aux prochaines parties.

Données et confidentialité

Toutes vos données (profils joueurs, historique, paramètres) sont stockées exclusivement sur votre appareil — IndexedDB pour les profils, localStorage pour les parties. Aucune donnée n'est envoyée à un serveur.

Installation (PWA)

Sur mobile ou tablette, utilisez Ajouter à l'écran d'accueil dans votre navigateur pour installer BarbuScore comme une application native, utilisable hors connexion.
